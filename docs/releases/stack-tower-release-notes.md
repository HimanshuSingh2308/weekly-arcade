## Features

### Core Gameplay
- One-tap timing mechanics (space/click/touch to drop)
- Blocks swing side-to-side with increasing speed
- Misaligned portions get trimmed
- Game over when tower width reaches minimum

### Scoring System
- Base: 10 points per block
- Perfect bonus: +15 points
- Combo multiplier: Up to 10x for consecutive perfects
- Height bonus: +5 per block in final score

### Visual Polish
- Gradient blocks with depth (highlights/shadows)
- Particle effects on perfect stacks
- Screen shake at high combos (15+)
- Gold flash effect for perfect stacks
- Falling piece particles when trimmed

### Mobile Optimization
- Touch anywhere to drop
- Responsive canvas sizing
- Input debouncing to prevent double-tap
- iOS AudioContext fix (single instance, user gesture init)

## Bug Fixes Applied

### During QA
1. **Spacebar Start** - Fixed empty if-block preventing spacebar from starting game
2. **Input Debouncing** - Added 50ms debounce to prevent double-firing on mobile
3. **RAF Cleanup** - Cancel requestAnimationFrame on game over
4. **Particle Limit** - Capped particles at 200 to prevent memory issues

### During Code Review
1. **AudioContext Leak** - Changed from creating new context per sound to single instance
2. **iOS Audio** - Added user gesture initialization for AudioContext
